# Den Perfekta Mackan

> Ett ambitiöst projekt för att vetenskapligt och metodiskt konstruera världens bästa smörgås.

## Overview
En fullstack-lösning för att lösa mänsklighetens mest pressande problem: hur gör man den perfekta mackan? Projektet kombinerar bröd-teknologi, påläggs-algoritmer och strukturell integritetsbedömning i ett sömlöst gränssnitt.

- Backend: Kök
